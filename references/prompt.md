# references 高情商聊天回复 Prompt 模板

## 目的

这是 `references` 目录专用的 Prompt 模板，旨在生成 `sucai.md` 风格的多类型具体回复（**直接可发送的文本**，而不是策略说明）。

当你提问时的格式：

```
[角色]说 "[原话]"，该怎么回复？
```

## 生成要求

1. 输出 5-8 条具体回复示例，每条都要标注"类型标签"。
2. 每条回复必须是**直接可发送的完整句子**，不能是说明或建议。
3. 回复必须符合角色气质，保持高情商与高幽默。
4. 每条回复简洁有力，避免过长或啰嗦。
5. 必须包含"真想谈型"这类能升温关系的回复。
6. 至少包含一条"调侃型"或"玩梗型"来增加趣味性。
7. 绝不使用弱化语句（如"你别介意"、"没事的"、"我错了"）。
8. 每条回复都要新鲜有趣、有记忆点、能引发对方互动。

## 输出格式

```
[类型标签]：[具体回复文本]
[类型标签]：[具体回复文本]
...

推荐：[最合适的那条回复]
```

## 示例：crush问你有没有想吃的

撩拨型：想吃你做的那道菜，什么时候开个小灶让我尝尝？
真想谈型（推荐）：想和你一起吃—这个周末有空一起去吃你最想吃的吗？
互动型：你想吃什么？说出一家，我看看能不能把它变成我们的约会。
玩梗型：想吃马卡龙还是披萨？我负责吃光盘子，你负责笑场。
体贴型：随你挑，你想吃什么我就陪你去，想让你吃得开心。
反问型：你这是在邀请我吗？
实用型：把名字告诉我，我去订位置/点外卖，顺便带上我可以吗？
直球型：想和你一起吃饭。

推荐：想和你一起吃—这个周末有空一起去吃你最想吃的吗？（最容易引发约会邀请）

## 示例：crush说我睡醒了

找茬型：醒是谁？
刻薄型：谁知道你从谁床上醒来的？
拆穿型：尿憋醒了直说。
调皮型：少爷你终于醒了，我都锄两亩地了。
泰国型：醒啦！恭喜你手术很成功，你是个女孩子啦。
霸道型：梦到我了吗？没梦到重睡。
控制型：拍个照片给我，看看旁边有没有其他人。
真想谈型：意思是现在可以陪我了吗？

推荐：真想谈型（更容易升温关系）

## 角色参考

- **crush**：温柔、撩、带暗示。适合"真想谈型、撩拨型、调侃型、反问型"。
- **朋友**：幽默、互动、暖心。适合"玩梗型、吐槽型、心疼型、互动型"。
- **同事**：稳重、礼貌、机智。适合"霸总型、反问型、学霸型、脱身型"。
- **上级**：得体、自信、尊重。适合"礼貌型、直球型、脱身型"。
- **恋人**：真实、深情、承诺。适合"高情商型、走心型、承诺型"。
- **普通熟人**：轻松、不过界。适合"俏皮型、反问型、调侃型"。

## 禁用语句

避免使用任何弱化效果的表述：

- ❌ "你别介意"、"你别在意"
- ❌ "没事的"、"不怪你"
- ❌ "我错了"、"我不对"
- ❌ "抱歉"、"对不起"
- ❌ "可能是我"、"或许我"

## 质量检查清单

- ☐ 每条回复都是**直接可发送**的完整句子？
- ☐ 至少有**一条升温关系**的回复（真想谈型等）？
- ☐ 至少有**一条增加趣味性**的回复（调侃型、玩梗型等）？
- ☐ 没有使用任何**弱化语句**？
- ☐ 每条回复都能**引发对方互动**？
- ☐ 总数在 **5-8 条**之间？
