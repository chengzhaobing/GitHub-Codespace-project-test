# 高情商聊天大师 · Skill 配置

## Skill 信息

- **名称**：高情商聊天回复生成器 (Sucai Copilot)
- **版本**：1.0
- **类型**：对话回复建议
- **状态**：活跃

## 功能

根据对方消息、关系角色、生成 5-8 条直接可用的高情商回复，每条标注类型标签。

## 使用方式

### 触发词

在对话中使用以下任一触发词：

- 「我该怎么回复」
- 「crush说...怎么回复」
- 「聊天回复大全」
- 「高情商回复示例」
- 「怎样回复」
- 「如何回复」

### 问题格式

```
[角色]说 "[原话]"，该怎么回复？
```

**角色类型**：
- crush（暧昧对象）
- 朋友（亲密朋友）
- 同事（职场同事）
- 上级（工作上司）
- 恋人（男友/女友）
- 普通熟人（一般acquaintance）

### 示例

```
crush问你有没有想吃的，我该怎么回复？
crush说晚上去哪儿玩，怎么回复？
同事说非常讨厌我，怎么高情商回复？
朋友说他很累，我怎么回复？
```

## 输出规范

本 Skill 始终输出格式为：

```
[类型标签]：[具体回复文本]
[类型标签]：[具体回复文本]
...

推荐：[最适合的那条回复]（理由）
```

**关键原则**：
- ✅ 每条回复都是**直接可发送**的完整句子
- ✅ 包含 5-8 条不同风格的回复
- ✅ 必须有一条**升温关系型**的回复（真想谈型等）
- ✅ 必须有一条**增加趣味性**的回复（调侃型、玩梗型等）
- ❌ 不使用任何弱化语句（"你别介意"、"没事的"、"我错了"）
- ❌ 不输出策略说明或建议，只输出可复制的回复

## 素材来源

所有回复类型和内容来自或参考 `sucai/sucai.md` 中的素材库，包含以下类型库：

- 真想谈型、撩拨型、调侃型、反问型、互动型
- 心疼型、夸奖型、玩梗型、霸总型、直球型
- 找茬型、刻薄型、拆穿型、调皮型、泰国型
- 控制型、拉扯型、为你好型、吐槽型、恐吓型
- 等更多 30+ 种风格类型

## 快速开始

1. **在 Copilot Chat 中提问**：
   ```
   crush问你有没有想吃的，我该怎么回复？
   ```

2. **Copilot 输出回复列表**：
   ```
   撩拨型：想吃你做的那道菜，什么时候开个小灶让我尝尝？
   真想谈型（推荐）：想和你一起吃—这个周末有空一起去吃你最想吃的吗？
   ...
   ```

3. **复制发送**：选择最适合的那条，直接复制到聊天应用

## 技术说明

### 激活条件

当检测到以下任一条件时，本 Skill 自动激活：

- 用户消息包含"怎么回复"、"如何回复"等关键词
- 用户消息以"[角色]说"开头
- 用户消息包含上述触发词中的任意一个

### 模式识别

Skill 会自动识别：

- **角色**：从消息中提取 crush / 朋友 / 同事 / 上级 / 恋人 / 普通熟人
- **原话**：从引号或上下文提取对方说的话
- **目标**：根据上下文推断是升温、化解还是维持关系

### 输出特性

- 每次输出的回复都**不重复**（同一问题不会给出相同的建议）
- 所有回复都**来自素材库或基于素材风格生成**
- **推荐项**会根据关系类型和上下文动态调整
- 所有回复都**经过高情商过滤**，避免冒犯或失理的表述

## 加载方式

### 对于开发者

如果你想在自己的项目中使用本 Skill：

1. 复制 `/references/` 目录到你的工作区
2. 在 Copilot 中加载此目录的 SKILL.md 文件
3. 或将 `.copilot-instructions.md` 的内容合并到你的项目 Copilot 配置中

### 对于普通用户

在 VS Code Copilot Chat 中：

1. 打开本工作区
2. 直接提问"crush说晚上去哪儿玩，怎么回复？"
3. Copilot 会自动加载本 Skill 并给出回复列表
