# 高情商聊天大师 · references Skill

> 这是一个落地型 Skill 文件，专门用于在 `/references` 目录中生成 `sucai.md` 风格的多种具体回复。

## 目标

- 根据用户提问，提取或生成类似 `sucai/sucai.md` 中的具体回复示例。
- 每个回复都带有"类型标签"（如心疼型、调侃型、真想谈型等）。
- 输出的是**直接可用的回复文本**，不是策略说明。
- 让使用者可以直接选择复制某一条回复来使用。

## 触发词

- 「我该怎么回复」
- 「crush说..怎么回复」
- 「聊天回复大全」
- 「高情商回复示例」
- 「怎样回复」

## 使用方式

当你提问时，格式为：
```
[角色]说 "[原话]"，该怎么回复？
```

例如：
- crush问你有没有想吃的，我该怎么回复？
- crush说我睡醒了，该怎么回复？
- 同事说讨厌我，怎么回复？

然后，本 Skill 会输出：
- 5-8 条具体回复示例
- 每条都标注「类型标签」（撩拨型、真想谈型、调侃型等）
- 直接可复制发送

## 输出格式

```
[类型标签]：[具体回复文本]
[类型标签]：[具体回复文本]
...

推荐：[最合适的那条回复]
```

## 示例输出

提问：crush说 "我睡醒了"，该怎么回复？

```
找茬型：醒是谁？
刻薄型：谁知道你从谁床上醒来的？
拆穿型：尿憋醒了直说。
调皮型：少爷你终于醒了，我都锄两亩地了。
泰国型：醒啦！恭喜你手术很成功，你是个女孩子啦。
霸道型：梦到我了吗？没梦到重睡。
控制型：拍个照片给我，看看旁边有没有其他人。
真想谈型：意思是现在可以陪我了吗？

推荐：真想谈型（更容易升温关系）
```

## 类型库

以下类别直接来源于 `sucai.md`，涵盖各种情境：

- 心疼型、夸奖型、换位思考型、安慰型、立刻行动型
- 真想谈型、玩梗型、环保型、霸总型、反问型、直球型
- 互动型、调侃型、祝福型、逆转型、找茬型、刻薄型
- 拆穿型、调皮型、泰国型、控制型、拉扯型、为你好型
- 吐槽型、恐吓型、公考型、疑惑型、找人型、羡慕型

## 推荐流程

1. 问题：提出 "[角色]说[原话]"
2. 本 Skill 输出：5-8 条具体回复
3. 推荐：标注最适合升温关系或达成目标的那条
4. 你选择：直接复制最喜欢的那条发送

## 重要原则

- 每条回复必须**直接可发送**，不能是建议或说明
- 回复必须有趣、有记忆点、能引发互动
- 必须包含能升温关系的选项（如真想谈型）
- 绝不使用弱化语句（如"你别介意"、"没事的"、"我错了"）
- 所有回复来自或参考 `sucai.md` 的素材库
