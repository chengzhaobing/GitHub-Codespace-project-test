## 决策的核心规则

### 规则1：情绪优先判断

**原则**：在任何判断之前，先判断对方的**当前情绪状态**

**判断方式**：

```
消息分析
   ↓
【Q】对方是什么情绪？
   /         |        \
消极        中立       积极
  ↓          ↓         ↓
走温暖     走反问     走调侃
路线       路线       路线
```

**实例对比**：

```
同样的消息："我最近很忙"

❌ 错误判断：
"哦，那咱们少聊点" （显得不关心）

✅ 正确判断：
情绪 = 消极（隐含）
→ 用温暖路线
→ 「听起来你压力很大」
```

---

### 规则2：关系阶段决定策略强度

**原则**：同样的场景，不同关系阶段用不同策略

**关系与策略的对应**：

```
初期关系
├─ 用"反问、调侃、直球"（测试和建立连接）
├─ 避免过度暖和
└─ 例：「是报备还是邀请？」

中期关系
├─ 用"真想谈、高情商、撩拨"（升级亲密）
├─ 温度和智商平衡
└─ 例：「只是没找你又不是没想你」

成熟关系
├─ 用"承诺、立刻行动、深情"（确认定位）
├─ 更多用行动而不是话
└─ 例：「我现在过来陪你」
```

**失败案例**：

```
❌ 初期关系用太深情的语言
  「我一直很在乎你」
  → 显得过度投入，反而没吸引力

❌ 成熟关系还用反问试探
  「是报备还是邀请？」
  → 显得不够确定，对方反感

✅ 正确做法：按关系阶段调整强度
```

---

### 规则3：目标明确化

**原则**：在选策略前，先问"我想要什么"

**三个主要目标**：

```
目标A：维持关系
├─ 适用策略：心疼、安慰、陪伴、学霸
├─ 节奏：慢、稳
└─ 示范：「听起来你挺累的」

目标B：升级关系
├─ 适用策略：真想谈、高情商、反问、邀约
├─ 节奏：进攻、递进
└─ 示范：「我明天也漂亮，明天见面吗？」

目标C：建立边界
├─ 适用策略：霸总、脱身、反讽、直球
├─ 节奏：冷静、坚定
└─ 示范：「个人情绪不用对我汇报」

目标D：化解尴尬
├─ 适用策略：脱身、自嘲、反讽
├─ 节奏：快、转移
└─ 示范：「想多了，帮我拿一下」
```

**决策树**：

```
你想要什么？
  /    |    |    \
维持  升级  边界  化解
 ↓     ↓    ↓    ↓
选   选   选   选
策   策   策   策
略   略   略   略
A    B    C    D
```

### 规则4：关系类型决定边界

**原则**：职场和恋爱关系的策略有明显区别

**职场关系规则**：

```
❌ 避免的策略：
  - 过度撩拨
  - 暴露个人情绪
  - 示弱
  - 长篇幅倾诉

✅ 推荐的策略：
  - 脱身（化解尴尬）
  - 学霸（展现能力）
  - 反问（掌控节奏）
  - 礼貌（保持距离）
```

**示范对比**：

```
场景：职场冲突

❌ 恋爱策略应用到职场：
「我们能不能好好聊一下，我其实是在乎你的」
→ 显得不专业，可能引起误会

✅ 职场化的回复：
「个人情绪不用对我汇报」
→ 既保持距离，又有尊严
```

---

### 规则5：信息充分度决定策略安全性

**原则**：信息越不足，用越保守的策略

**信息充分度分类**：

```
信息充分（你充分了解对方）
├─ 可用创意组合
├─ 可用风险较高的策略
└─ 例：「只是没找你又不是没想你」

信息一般（你有一定了解）
├─ 用通用策略
├─ 避免过度创新
└─ 例：「听起来你很累啊」

信息不足（你不太了解对方）
├─ 用最保险的策略
├─ 用心疼/安慰/反问
└─ 例：「我在呢」
```

**失败案例**：

```
❌ 信息不足还用高难度策略
  第一次聊天就用「高情商型」
  → 容易显得生硬或显得太熟悉

✅ 正确做法：
  信息不足 → 用温暖 + 反问
  慢慢积累信息 → 逐步升级策略
```

## 场景识别规则

### 规则1：自我贬低场景（S01）的识别

**特征词**：
```
「感觉」「不够」「怎么办」
「我长得」「我没有」「我这样」
```

**应对规则**：
```
Step 1：承认他的感受（心疼型）
Step 2：用反向思维回应（反讽型）
Step 3：升级（如果需要）

示范流程：
「谁长这样都不好受」（承认）
↓
「至少你感觉是对的」（反讽）
↓
「你在质疑我的眼光吗？」（升级）
```

---

### 规则2：状态通知场景（S02）的识别

**特征**：
```
简短消息 + 通知性质 + 可能的邀请意味
例：「我去洗澡了」「我要睡觉了」
```

**应对规则**：
```
Step 1：判断邀请意味
  ├─ 有邀请 → 反问「是报备还是邀请？」
  └─ 无邀请 → 简短回应「好的」

Step 2：根据亲密度调整
  ├─ 高亲密 → 升级为承诺「先说晚安的...」
  └─ 低亲密 → 展现能力「有不会的问我」
```

## 行为模式识别

### 模式1：对方在"试探"

**识别信号**：
```
1. 提问而不是陈述
   「你撤回了什么？」（而不是「我很在乎」）
   
2. 模糊表态
   「我最近很忙」（而不是「我不想和你聊」）
   
3. 观察你的反应
   快速发来多条消息
   看你多久回复
```

**对应规则**：
```
检测到"试探" 
  ↓
用"反问"而不是"直接回答"
  ↓
掌控话题节奏
  ↓
让对方主动表达更多
  ↓
你获得信息优势
```

**示范**：

```
对方：「你最近怎么都没找我？」（试探）

❌ 错误回应：
「我最近工作太忙了」
→ 落入了对方的试探陷阱

✅ 正确回应：
「想我了还是怪我了？」（反问）
→ 反向掌控，了解对方真实想法
```

---

### 模式2：对方在"示弱"

**识别信号**：
```
1. 用「我」开头
   「我最近很难受」
   
2. 期待被安慰
   话语中隐含求助信号
   
3. 观察你是否回应
```

**对应规则**：
```
检测到"示弱"
  ↓
用"温暖策略"而不是"逻辑分析"
  ↓
表达理解和陪伴
  ↓
可以逐步升级为支持
  ↓
建立情感连接
```

**示范**：

```
对方：「我最近工作太累了」（示弱）

❌ 错误回应：
「那就多休息，不用担心」
→ 显得敷衍，没有陪伴感

✅ 正确回应：
「听起来你压力很大，需要个人陪吗？」
→ 表达理解 + 提供陪伴
```

---

### 模式3：对方在"挑战"

**识别信号**：
```
1. 直接表达负面评价
   「我讨厌你」「你真讨厌」
   
2. 测试你的底线
   看你会不会生气或退缩
   
3. 期待你的反应
```

**对应规则**：
```
检测到"挑战"
  ↓
用"自信策略"而不是"防守"
  ↓
显得你有底线
  ↓
不被情绪控制
  ↓
掌控对话权力动态
```

**示范**：

```
对方：「我讨厌你」（挑战）

❌ 错误回应：
「你怎么能这样说，我对你那么好」
→ 显得被激怒，失去理性

✅ 正确回应：
「个人情绪不用对我汇报」
→ 冷静、有底线、显得成熟
```

## 决策的安全等级

### 等级1：最安全的回复

**特点**：任何场景都能用，失败率<10%

```
心疼型：「我能理解你的感受」
夸奖型：「你这一点真的很值得肯定」
安慰型：「有我在呢」
学霸型：「有不会的问我」
```

**使用场景**：
- 📍 初学者必用
- 📍 信息不足时
- 📍 不确定对方想什么时

---

### 等级2：中等风险的回复

**特点**：大多数场景能用，失败率10-30%

```
真想谈型：「我想听真话」
反问型：「是报备还是邀请？」
调侃型：「至少你感觉是对的」
脱身型：「想多了，帮我拿一下」
```

**使用条件**：
- 📍 有一定了解对方
- 📍 确定关系和目标
- 📍 已经掌握表达DNA

---

### 等级3：高风险的回复

**特点**：特定场景才能用，失败率30-50%

```
高情商型：「喜欢你已经超过2分钟了」
霸总型：「个人情绪不用对我汇报」
撩拨型：「可以一起洗吗？」
承诺型：「先说晚安的明早要先说早安」
```

**使用条件**：
- 📍 充分了解对方
- 📍 关系已经确立
- 📍 掌握了"表达DNA"的细节

## 决策的最终检查清单

在发送任何回复前，问自己这9个问题：

```
□ 1. 我准确判断了对方的情绪吗？
□ 2. 我清楚自己的目标是什么吗？
□ 3. 我的策略符合当前关系阶段吗？
□ 4. 如果是职场关系，我有没有过度撩拨？
□ 5. 我掌握了对方足够的信息吗？
□ 6. 这个场景适合这个策略吗？
□ 7. 我的表达方式自然吗？
□ 8. 我有没有踩到禁忌？
□ 9. 对方看到这条消息会是什么反应？
```

**全部打勾✅** → 可以发送  
**有2个以上打×❌** → 再想想  
**超过3个打×❌** → 先不发，等想清楚

## 决策的优化循环

```
第1次聊天
  ↓
按照系统选择策略
  ↓
记录对方反应
  ↓
成功 ✅ → 记住这个方案，下次继续用
  ↓
失败 ❌ → 分析原因，下次改进
  ↓
第2次聊天
  ↓
基于上次经验调整
  ↓
继续优化
```

**关键**：每一次都是学习机会，不要重复同一个错误
